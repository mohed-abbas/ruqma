## Image Guidelines

- **Format**: JPG or WebP for photographs
- **Optimization**: Compress images for web (use tools like ImageOptim, TinyPNG)
- **Quality**: High-quality images that reflect luxury retail environments
- **Subject**: Images should showcase professional, upscale retail spaces
- **Color Palette**: Align with brand colors (gold #d4af37, black, white)
- **Aspect Ratio**: Maintain 4:3 ratio for consistency across all store cards

## Placeholder Images

Until final images are ready, you can use:
- Stock photos from Unsplash, Pexels, or similar
- Temporary luxury retail environment photos
- Generic upscale storefront images

Make sure to replace placeholders with actual partner store photography before production deployment.

## Adding New Stores

When adding a new partner store:
1. Add store data to `data/content/where-to-buy.json`
2. Create appropriately sized image (1200×900px, 4:3 ratio)
3. Name file based on store ID (e.g., `new-store-name.jpg`)
4. Add to this directory
5. Update this README with new image specifications
