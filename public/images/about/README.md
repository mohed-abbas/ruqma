# Our Story Page Images

This directory contains images for the Our Story (`/our-story`) page.

## Required Images

### 1. story-intro.jpg
- **Purpose**: Story Introduction section background
- **Recommended Size**: 1200×900px (4:3 aspect ratio)
- **Description**: Image showcasing Ruqma craftsmanship detail or product close-up
- **Alt Text**: "Ruqma craftsmanship detail showcasing our dedication to quality"

### 2. craftsmanship.jpg
- **Purpose**: Full-width craftsmanship section background
- **Recommended Size**: 1920×1080px (16:9 aspect ratio)
- **Description**: Image of artisan at work or manufacturing process
- **Alt Text**: "Ruqma craftsmanship process showing artisan at work"

## Image Guidelines

- **Format**: JPG or WebP for photographs
- **Optimization**: Compress images for web (use tools like ImageOptim, TinyPNG)
- **Quality**: High-quality images that reflect luxury brand aesthetic
- **Subject**: Images should showcase quality, craftsmanship, and attention to detail
- **Color Palette**: Align with brand colors (gold #d4af37, black, white)
